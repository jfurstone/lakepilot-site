# LakePilot — site

Public marketing landing page for **LakePilot** (a hands-on tutor for building AI
agents on Databricks). The course source lives in a separate private repo; this repo
holds only the landing page and an in-browser demo of the two-pane app.

- `index.html` — the landing page
- `app/` — the live two-pane demo (lessons + a demo-mode sandbox), embedded by the landing page

Published via GitHub Pages.
